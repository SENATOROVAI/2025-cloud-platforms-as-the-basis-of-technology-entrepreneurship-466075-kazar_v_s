# 2. Краткий обзор Google Cloud Platform

Google Cloud Platform (GCP) — это облачная платформа, предоставляющая инфраструктурные и платформенные сервисы для разработки, развертывания и масштабирования приложений.

Основные преимущества GCP:

- Гибкая масштабируемость ресурсов  
- Глобальная инфраструктура  
- Интеграция сервисов  
- Гибкая система управления доступом (IAM)  
- Оплата по модели Pay-as-you-go  

В рамках лабораторной работы были исследованы следующие сервисы:

- Identity and Access Management (IAM)  
- Compute Engine  
- Cloud Storage

## 5.4 Копирование файлов из Cloud Storage

Создание директории и копирование файлов:
```bash
mkdir lab1_files
cd lab1_files
gsutil cp gs://lab1-bucket-itmo/* .
ls -lah
```
После выполнения команды файлы pic1.jpg, pic2.jpg и pic3.jpeg успешно скопированы на виртуальную машину.

## 5.5 Проверка ограничений доступа (IAM)
После изменения роли сервисного аккаунта на Compute Viewer была выполнена проверка доступа к IAM-политике бакета:

```bash
gsutil iam get gs://lab1-bucket-itmo
```

Результат выполнения:

AccessDeniedException: 403 Provided scope(s) are not authorized

Это подтверждает, что сервисный аккаунт больше не имеет прав администратора Cloud Storage
