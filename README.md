# Облачные платформы как основа технологического предпринимательства

**Репозиторий лабораторных работ**

## Описание

Данный репозиторий содержит лабораторные работы, выполненные в рамках дисциплины
**«Облачные платформы как основа технологического предпринимательства»**.

Основная цель работ — освоение практических навыков работы с облачными сервисами, развёртывания инфраструктуры, настройки сервисов и интеграции приложений в облачной среде.

Каждая лабораторная работа включает:

* подробный отчёт (`labX_report.md`)
* иллюстрации и скриншоты (`images/`)
* дополнительные конфигурационные файлы (при необходимости)

## Содержание лабораторных работ

* **Lab 1** — базовая настройка облачной инфраструктуры
* **Lab 2** — развёртывание сервисов и конфигурация окружения
* **Lab 3** — интеграция облачных сервисов и автоматизация
* **Lab 4** — практическая реализация облачного решения и анализ результатов

---

## Используемые технологии

В лабораторных работах применяются:

* облачные платформы (IaaS / PaaS)
* контейнеризация и сервисы развертывания
* инструменты автоматизации инфраструктуры
* системы мониторинга и управления
